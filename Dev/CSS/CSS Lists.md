在 HTMl 中有两种列表
- 有序列表 `ol`
- 无序列表 `ul`

### 不同的列表项标记
可以通过 `list-style-type` 属性来指定，列表标记的符号
- `circle`
- `square`
- `upper-roman`
- `lower-alpha` (阿拉伯字母)

`list-style-position` 来指定符号的位置
- `outside`
- `inside`

### 移除默认的样式
`list-style-type:none` 可以移除列表符号（<ul>,<ol>）的样式，默认是是带有 margin 和 padding 的
